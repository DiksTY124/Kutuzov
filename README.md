## О проекте

Сайт включает в себя информацию о жизни и деятельности Кутузова, его значении в истории России и основных достижениях. Проект выполнен с акцентом на современный дизайн и удобство навигации, что позволяет пользователям легко находить интересующие их сведения.

## Используемые технологии

- HTML
- CSS

## Как запустить

Для просмотра сайта достаточно открыть файл index.html (Путь к файлу: main/index/html) в любом веб-браузере.
